# 算法动画
> 冒泡、选择、插入、快速、归并

| 排序算法 (Algorithm) | 平均时间复杂度 (Average) | 最坏时间复杂度 (Worst) | 最好时间复杂度 (Best) | 空间复杂度 (Space) | 稳定性 (Stability) | 原地性 (In-place) |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **冒泡排序** (Bubble) | $O(n^2)$ | $O(n^2)$ | $O(n)$ | $O(1)$ | **稳定** | Yes |
| **选择排序** (Selection)| $O(n^2)$ | $O(n^2)$ | $O(n^2)$ | $O(1)$ | 不稳定 | Yes |
| **插入排序** (Insertion)| $O(n^2)$ | $O(n^2)$ | $O(n)$ | $O(1)$ | **稳定** | Yes |
| **希尔排序** (Shell) | $O(n^{1.3})$ (依赖步长) | $O(n^2)$ | $O(n)$ | $O(1)$ | 不稳定 | Yes |
| **归并排序** (Merge) | $O(n \log n)$ | $O(n \log n)$ | $O(n \log n)$ | $O(n)$ | **稳定** | No |
| **快速排序** (Quick) | $O(n \log n)$ | $O(n^2)$ | $O(n \log n)$ | $O(\log n)$ | 不稳定 | Yes |
| **堆排序** (Heap) | $O(n \log n)$ | $O(n \log n)$ | $O(n \log n)$ | $O(1)$ | 不稳定 | Yes |
| **计数排序** (Counting)| $O(n+k)$ | $O(n+k)$ | $O(n+k)$ | $O(k)$ | **稳定** | No |
| **桶排序** (Bucket) | $O(n+k)$ | $O(n^2)$ | $O(n)$ | $O(n+k)$ | **稳定*** | No |
| **基数排序** (Radix) | $O(n \times k)$ | $O(n \times k)$ | $O(n \times k)$ | $O(n+k)$ | **稳定** | No |
